# Oge-# 📚 Чек-лист подготовки к ОГЭ

Интерактивный чек-лист для подготовки к ОГЭ по 4 предметам:
- 📐 Математика (20 заданий)
- 🧪 Химия (23 задания)
- 📝 Русский язык (15 заданий)
- ⚡ Физика (22 задания)

## ✨ Функции
- ✅ Отметка выполненных заданий
- 🎲 Случайный выбор
- 📊 Статистика и календарь
- 📝 Заметки к заданиям
- 💌 Письма поддержки от аниме-персонажей
- 📈 Отслеживание пробников
- 😈 Режим Злодея / 🌙 Ночной режим
- 📥 Экспорт/импорт данных

## 🚀 Запуск
Просто открой `index.html` в браузере!
